# Shihab | Decentralized Infrastructure Engineer
**Computer Science & Engineering | Focus: Solana DeFi & RWA Architecture**

Architecting deterministic state machines and collateralized lending infrastructure. Specializing in Rust-based memory-safe execution, high-throughput Solana on-chain programs, and secure atomic settlements for neo-banking environments.

---

### ⚙️ Core Operational Stack
*   **On-Chain Execution:** Solana, Rust, Anchor Framework, Native Instruction Serialization
*   **Off-Chain Routing:** Actix-web, Next.js, TypeScript
*   **System Environment:** Pop!_OS (Linux), Deterministic CI/CD Pipelines
*   **Asset Architecture:** JIT-funded stablecoin corridors (EURC, USDC), RWA integrations, NFC-to-fiat bridging

---

### 🏗️ Primary Infrastructure Deployments

**1. anchor-lend**
> Collateralized lending and borrowing monorepo. Engineered for strict memory efficiency and instant liquidation corridors.

**2. CryptoCardBridge**
> Integrated lending vault and crypto payment layer. Facilitates Just-in-Time (JIT) funded debit card transactions with real-time atomic settlement.

**3. sol-wallet-raw**
> Bare-metal Solana wallet program. Bypasses the Anchor framework entirely to execute native instruction serialization and enforce manual account memory boundaries.

**4. telemetry_oracle**
> Rust-based local price oracle CLI. Implements defensive sanitization layers for secure off-chain to on-chain asset evaluation and telemetry ingestion.
